# E-commerce-Sales-Analysis In Python Using Pandas, Numpy,Seaborn & Matplotlib
Conducted exploratory data analysis on 10,000+ e-commerce transactions to identify revenue drivers
Regional performance, and customer purchasing patterns
Discovered that Electronics was the highest-performing category, UPI was the preferred payment method, and higher discount levels significantly reduced profitability.
Recommended optimizing discount strategies, strengthening sales efforts in underperforming regions, and prioritizing inventory for top-selling products to improve business performance.

✅ Monthly Sales Trends
✅ Region-wise Sales Analysis
✅ Category wise Sales Analysis
✅ Top 10 Customers with Highest Sales
